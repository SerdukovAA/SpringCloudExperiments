Это конфигурационный сервер.
Это REST сервер который отдает насройки по специфическим запроса. Т.е приложение которое жлает получить свой конфиг,
делате запрос к этому серверу.

Пример запроса:
http://localhost:8888/config-file/default
В данном случае приложение "клиент-до-настроек" передает config-file как свое "имя".
